Downloading and installing

Click the button below, and you'll get the .zip of the game. After that you can choose if start the game with an IDE or using the CLI and digit the following command:

First Step

The main menu of Jungle Temple. You can choose if play a new game or load a game typing the name of the saving

The commands

All commands and moves you can do simply writing them on the CLI' game

Move

Typing move and the direction (north, east, west, sud) you prefer you can change the room. Also, simply typing back you return to the previous room

Item Interact

Look, take and drop allow you to see all items and entities in the room, take object/weapon and drop object/weapon. Typing the command inventory you can see all the items you have in your bag!

Entities Interact

Talk and give allow you to receive information and objects from traders

Combat

Attack and parry allow you to inflict damage to the monsters or avoid a possible attack from them
