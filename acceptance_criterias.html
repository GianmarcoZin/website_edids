Acceptance Criterias

#1
Configuration

AS A player I WANT to be able to save, exit, load the game SO THAT I can log off without losing any progress.

#2
Move

AS A player I WANT access to movement commands SO THAT I can navigate between rooms of the map.

#3
Item Interaction

AS A player I WANT access to item commands SO THAT I can interact with items in a room and in my inventory

#4
Stats

AS A player I WANT to view my statistics SO THAT I can keep track of the character state (health, inventory weight, score).
